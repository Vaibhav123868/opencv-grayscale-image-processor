# OpenCV Grayscale Image Processor

A simple image processing project built using Python and OpenCV.
This program allows users to upload an image, convert it into grayscale, and choose whether to display the processed image or save it to disk.

----------------------------------------------------------------------------------------------------------

## Project Overview

This project demonstrates basic image processing using the OpenCV library in Python. The program reads an image from the user, converts it into grayscale, and then gives the user two options:

* Display the processed image on the screen
* Save the processed image to a new file

This project is useful for beginners who want to learn Python image processing and OpenCV basics.

------------------------------------------------------------------------------------------------------------

## Features

* Upload an image using a file path
* Convert a color image to grayscale
* Display the processed image
* Save the processed image
* Simple command line interface

## How to Run the Project

1. Install Python on your system.

2. Install OpenCV:

```
pip install opencv-python
```

3. Run the program:

```
python main.py
```

4. Provide the image path when prompted. Example:

```
Upload the image: dog.jpg
```

5. Choose an option:

```
show  - Display the grayscale image
save  - Save the grayscale image
```
